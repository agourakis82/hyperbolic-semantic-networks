## Release Contents

This release contains the complete dataset, analysis code, and results for the manuscript:

**"Consistent Evidence for Hyperbolic Geometry in Semantic Networks Across Four Languages"**

Submitted to: *Network Science* (Cambridge University Press)

## Files Included

### 1. Processed Data (`data/processed/`)
- `spanish_edges.csv` - Spanish SWOW network (500 nodes)
- `english_edges.csv` - English SWOW network (500 nodes)
- `dutch_edges.csv` - Dutch SWOW network (500 nodes)
- `chinese_edges.csv` - Chinese SWOW network (500 nodes)

Format: `source,target,weight` (directed weighted edges)

### 2. Analysis Results (`results/structural_nulls/`)
- Configuration model nulls (4 languages, M=1000 each)
- Triadic-rewire nulls (2 languages, M=1000 each)
- Complete statistics: Δκ, p_MC, Cliff's δ, CI 95%

### 3. Analysis Code (`code/`)
- `preprocess_swow_to_edges.py` - Data preprocessing
- `07_structural_nulls_single_lang.py` - Null model generation (FIXED bugs)
- `08_fill_placeholders.py` - Manuscript value injection
- `deep_insights_miner.py` - Statistical insights mining
- `requirements.txt` - Python dependencies

### 4. Manuscript
- `manuscript_v1.8.12_FINAL.pdf` - Submitted version
- `supplementary_materials.pdf` - Complete supplements

## Key Results

- **3 of 4 languages** show robust hyperbolic geometry (p < 0.001)
- **Effect homogeneity:** I²=0% across languages
- **Perfect separation:** |Cliff's δ| = 1.00 for all significant tests
- **Triadic validation:** 51-59% variance reduction vs. configuration
