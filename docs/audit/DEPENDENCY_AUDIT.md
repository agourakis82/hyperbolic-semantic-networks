## Critical Updates Needed

### NumPy 2.0 Migration

**Breaking Changes**:
- `np.int` → `np.int64` (removed)
- `np.float` → `np.float64` (removed)
- String dtype rewritten (10x performance improvement)
- New dtype API

**Impact**:
- Medium risk: Code uses deprecated aliases
- Requires code changes
- Significant performance benefits

**Recommendation**:
- Update to NumPy 2.0+ before migration
- Fix all deprecated usage
- Test thoroughly
