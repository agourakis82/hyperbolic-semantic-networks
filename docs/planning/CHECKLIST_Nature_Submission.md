# Nature Communications Submission Checklist

## Main Manuscript
- [x] Word count ≈ 4,500 (main text)
- [x] Sections (Intro/Methods/Results/Discussion) in Nature style
- [x] Clinical findings integrated (Sections 3.4–3.6)
- [x] References updated (Q9–Q14 papers added)
- [x] Data & Code Availability statements present
- [x] Acknowledgments / Competing Interests compliant

## Figures & Tables
- [x] Figures (≥300 dpi) with expanded legends
- [x] Color palettes consistent with Nature guidelines
- [x] Supplementary figure plan (meta-analytics, clinical visuals)
- [ ] Final review of figure numbering and cross-references

## Supplementary Materials
- [ ] Prepare PDF with extended methods, null model details, meta-analysis protocols
- [ ] Include tables for clinical datasets and additional stats
- [ ] Provide scripts (or link) in repository with version tag

## References & Compliance
- [x] `references.bib` updated with new citations
- [x] DOIs verified (where available)
- [ ] Cross-check in-text citations vs. references list
- [ ] Confirm reference count within journal limits

## Submission Package
- [ ] Export main manuscript to PDF for final review
- [ ] Create cover letter draft outlining key contributions
- [ ] Prepare Highlights/Graphical abstract (if required)
- [ ] Complete journal checklists (ethical compliance, data policies)
